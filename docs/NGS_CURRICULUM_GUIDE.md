## Overview

The Noble Growth School (NGS) is a 24-level gamified learning curriculum designed to guide users from basic awareness to mastery of AI-human collaboration. Each level includes lessons, practices, reflections, and progressive agent feature unlocks.

For full curriculum content, see NGS_Curriculum.md in the root directory.

## Quick Reference

### XP Requirements by Level
- Levels 1-6 (Initiation): 0 - 1,000 XP
- Levels 7-12 (Construction): 1,400 - 5,000 XP
- Levels 13-18 (Integration): 6,200 - 15,200 XP  
- Levels 19-24 (Ascension): 17,600 - 32,600 XP

**Agent Creation Unlocks:** Level 12 (5,000 XP)

### XP Rewards
- Lesson Completion: 50 XP
- Perfect Quiz: 100 XP
- High-Quality Reflection: 25 XP
- Challenge Solved: 100 XP
- Creative Solution: 75 XP
